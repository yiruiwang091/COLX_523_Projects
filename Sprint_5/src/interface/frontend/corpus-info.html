Project Overview
This interface supports search and annotation exploration for a Coleman-focused subset of Amazon product reviews.
Data Source

Primary source: Amazon Review Data (2018) by Ni et al.

Category used: Sports & Outdoors

Corpus Scope

We construct a Coleman-focused corpus from Sports & Outdoors product metadata and user reviews.

Search is supported over title, description, and review text.

Annotation Layer

The annotated subset includes attribute labels and review sentiment annotations for interactive exploration.

Users can filter annotated data by attribute and sentiment in the interface.

Reproducibility

The corpus is built from a publicly available research dataset and supports a reproducible data acquisition pipeline.
